# Threadline

## Package Manager

Use `bun` — never `npm` or `yarn`.

- `bun install` for dependencies
- `bun run dev` for dev server
- `bun run build` for production build
- `bunx` instead of `npx`

## Git Workflow

- All new PRs target `dev`, not `main`. Create feature branches from `dev` and open PRs back into `dev`.
